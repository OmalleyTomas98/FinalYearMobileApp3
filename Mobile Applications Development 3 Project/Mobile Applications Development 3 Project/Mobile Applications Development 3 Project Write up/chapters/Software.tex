	\item Unity Engine  
   \begin{itemize}
     \item Unity is a cross-platform game engine developed by Unity Technologies, first announced and released in June 2005 at Apple Inc.'s Worldwide Developers Conference as a Mac OS X-exclusive game engine. As of 2018, the engine had been extended to support more than 25 platforms.


   \end{itemize}


\item LaTex Document Editor
   \begin{itemize}
     \item LaTeX, stylized within the system as LaTeX, is a software system for document preparation. When writing, the writer uses plain text as opposed to the formatted text found in "What You See Is What You Get" word processors like Microsoft Word, LibreOffice Writer and Apple Pages
   \end{itemize}
